## Employe api automation with newman
## About this project
This is a employee api automation project.it is facilated to user create employee ,modify employee,get employee data and delete.We have also create api automation and generate newwman report.
## Prerequisite
- nodejs
## How to run this project
- clone the project [Employee APi project](https://github.com/khundakerfaisal/EmployeApiAutomationWithNewmanReport.git)

-  ```Hit the command```
 ``` bash
npm init -y
```
``` bash
npm i newman
```
